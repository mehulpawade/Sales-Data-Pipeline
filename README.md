# Motorola Sales Analytics Dashboard

## 📊 Project Overview
This project showcases an interactive Power BI dashboard developed to analyze Motorola mobile sales performance. The dashboard provides insights into sales trends, product performance, and customer purchasing patterns through dynamic visualizations.

## 🚀 Dashboard Features

- KPI cards displaying:
  - Mobile Model
  - Brand
  - Payment Method
  - Day-wise Information

- Interactive slicers for dynamic filtering and analysis.
- Line chart showing total quantity sold by month.
- Global map visualization displaying city-wise sales distribution.
- Brand-wise performance table including:
  - Total Sales
  - Total Quantity Sold
  - Total Transactions
- Month selector for monthly sales analysis.
- Mobile model-wise sales visualization.

## 🛠️ Tools & Technologies
- Power BI
- Data Analysis
- Python
- SQL

## 🎯 Objective
To transform raw sales data into meaningful insights and support data-driven decision-making through an interactive and user-friendly dashboad.

## 📌 Key Insights
- Track monthly sales performance.
- Analyze sales distribution across different cities.
- Compare brand performance based on sales, quantity, and transactions.
- Identify top-performing mobile models and payment methods.
